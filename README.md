# Fast Food Sales Analysis

## Project Overview
This project analyzes a fast food sales dataset to understand customer behavior, sales trends, and business performance using Python and Power BI.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- DAX

## Key Metrics
- Total Sales
- Total Orders
- Average Order Value
- Total Quantity Sold

## Insights
- Peak sales occur during Evening and Night.
- Certain items contribute significantly to revenue.
- Fast Food being the most preferred category among consumers
- Sandwich generated the highest revenue, while cold coffee recorderd
  the highest sales volume.
- Customers often purchase multiple items per order.
- Cash payments are more common than digital.
